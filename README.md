# Advanced Scientific Calculator

A modern web-based scientific calculator developed using Python Flask. The application supports advanced mathematical operations, trigonometric functions, logarithms, calculation history, theme customization, and degree/radian mode.

## Features

- Basic arithmetic operations
- Scientific functions
  - Sine (sin)
  - Cosine (cos)
  - Tangent (tan)
  - Square Root (sqrt)
  - Logarithm (log)
  - Natural Logarithm (ln)
  - Power Functions
- Degree and Radian mode
- Calculation history
- Dark and Light theme support
- Keyboard input support
- Responsive user interface
- Secure expression evaluation

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
